# Verilog-implementation-of-pipelined-MIPS32-processor
A Verilog-based pipelined MIPS32 processor implementing a five-stage pipeline (IF, ID, EX, MEM, WB). Supports basic MIPS32 instructions (add, sub, addi, mul, shift left etc.), with testbench simulation for verification. Features hazard detection and data forwarding mechanisms.
